# StylePicker 下拉化设计

## 概述

将 StylePicker 组件从"5 个按钮 + 独立输入框"改为一个下拉选择框（`<select>`），包含 5 个预设风格选项和一个"自定义..."选项。

## 交互逻辑

1. 下拉框默认显示当前选中的风格（或"自定义..."）
2. 选择预设风格 → 清空自定义输入，设置 `selectedStyle`
3. 选择"自定义..." → 显示输入框，清空 `selectedStyle`
4. 在输入框中输入内容 → 设置 `customStyle`

## 样式

- 下拉框样式与现有输入框保持一致（圆角、边框、聚焦效果）
- 输入框默认隐藏，选择"自定义..."时显示

## 文件变更

仅修改 `src/components/StylePicker.vue`，其他文件不变。
